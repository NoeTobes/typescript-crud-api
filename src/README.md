# TypeScript CRUD API

A fully typed REST API built with TypeScript, Node.js, Express, and JSON file storage. This project demonstrates CRUD operations, JWT authentication, role-based access control, and input validation.

## Features

- ✅ **TypeScript** for type safety and better developer experience
- ✅ **JWT Authentication** for secure user login
- ✅ **Role-Based Access** (Admin/User) with TypeScript enums
- ✅ **Input Validation** using Joi
- ✅ **Error Handling Middleware** for consistent error responses
- ✅ **JSON File Storage** - No database installation required!
- ✅ **CRUD Operations** - Create, Read, Update, Delete users
- ✅ **Fully Typed Code** - Interfaces, enums, and type definitions throughout

## Prerequisites

- **Node.js** (v14 or higher) - [Download](https://nodejs.org/)
- **npm** (v6 or higher) - comes with Node.js
- **Git** (optional) - for version control

No database installation needed! The API uses a simple JSON file for data persistence.
